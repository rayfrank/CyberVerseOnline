
# CYBERVERSE (FINAL)
### AI-Enhanced Cybersecurity Learning Metaverse (Unity)

CYBERVERSE is an **AI-powered, gamified cybersecurity learning platform** built in **Unity 6.2.7f2**.  
It recreates a fully interactive **3D cyber lab** where learners explore, interact with tools, and complete real-world cybersecurity missions inside a safe simulated environment.

This project was developed as a **finalist solution for the NIRU AI Hackathon**, under the thematic area **Cybersecurity & Data Protection**, aligned to the theme **AI for National Prosperity**.

---

## 🎯 Problem Statement
Kenya’s rapid digital transformation across government, education, finance, and private institutions has led to a growing demand for cybersecurity skills. However, most institutions still rely on **theoretical teaching methods**, lacking access to cyber ranges, labs, and practical environments.

As a result:
- Learners understand concepts but cannot apply them
- Cyber threats such as phishing, device compromise, and scams continue to rise
- A national cybersecurity skills gap persists

CYBERVERSE bridges this gap through **experiential learning powered by AI**.

---

## 💡 Proposed Solution
CYBERVERSE provides:
- A walkable **3D cyber lab** where learners complete missions instead of memorizing notes
- A conversational AI mentor (**NURU AI**) embedded inside the environment
- Gamified progression through real-world cybersecurity scenarios

Learners interact with AI terminals, receive step-by-step guidance, analyze threats, and learn defensive techniques in a controlled, risk-free space.

---

## ⭐ Key Features
- **Immersive 3D Cyber Lab**
  - Walk, explore, and interact with cybersecurity tools
- **Gamified Cybersecurity Missions**
  - Phishing analysis, password security, device hardening (MVP scope)
- **NURU AI Assistant**
  - Context-aware guidance, explanations, hints, and feedback
- **Kenyan Context Awareness**
  - Scenarios based on common local threats (e.g. WhatsApp scams, mobile-money phishing)
- **Modular Design**
  - Easily expandable to advanced topics (network security, ransomware, cloud security)
